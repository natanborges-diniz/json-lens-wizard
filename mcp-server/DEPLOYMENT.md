# Deployment

## Recomendação
Usar Railway primeiro pela velocidade.

## Passos
1. subir a pasta `mcp-server` para um repositório
2. criar novo projeto no Railway
3. apontar para essa pasta
4. deploy automático
5. validar `/health`
6. usar a URL no Lovable

## Endpoint inicial
- `/health`
- `/project-status`
- `/normalized-data`
- `/supplier-docs`

## Fase seguinte
Substituir este servidor HTTP mínimo por servidor MCP remoto completo.
